# 🔧 Render.com 环境变量配置

## 📋 必需的环境变量
复制以下变量到 Render 的 Environment 设置中：

### 🌍 基础环境配置
ENVIRONMENT=production

### 🔐 JWT 安全配置
SECRET_KEY=cortex-ai-workspace-jwt-secret-2024
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30

### 🤖 OpenAI API 配置
OPENAI_API_KEY=你的OpenAI密钥

### 🗄️ 数据库配置
DATABASE_URL=postgresql://用户名:密码@主机:端口/数据库名

## 📝 配置步骤

### 1. 获取 OpenAI API 密钥
- 访问: https://platform.openai.com
- 注册/登录账号
- 创建 API 密钥
- 复制密钥到 OPENAI_API_KEY

### 2. 配置数据库连接
- 在 Render 创建 PostgreSQL 数据库
- 复制连接字符串
- 粘贴到 DATABASE_URL

### 3. 在 Render 中添加变量
- 进入 Web Service 设置
- 点击 "Environment" 标签
- 逐个添加上述变量

## ⚠️ 重要提示
- 不要在代码中硬编码密钥
- DATABASE_URL 会自动从数据库服务获取
- 确保所有变量都正确设置

## 🔍 验证配置
部署完成后访问: https://your-app.onrender.com/docs
应该能看到 FastAPI 文档页面
